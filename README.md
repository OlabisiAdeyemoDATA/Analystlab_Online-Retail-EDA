# Online Retail Data Cleaning and Exploratory Data Analysis

## Project Overview

This project focuses on cleaning, exploring, and analyzing an online retail transaction dataset using Python. The objective was to understand sales patterns, customer purchasing behavior, product performance, and revenue trends.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Project Tasks

- Dataset Understanding
- Data Cleaning
- Missing Value Treatment
- Duplicate Removal
- Data Standardization
- Data Validation
- Exploratory Data Analysis
- Data Visualization
- Business Insights

 ## Key Insights

- The United Kingdom generated the highest revenue, contributing over £9 million in sales.
- November 2011 recorded the highest monthly revenue.
- PAPER CRAFT, LITTLE BIRDIE was the most purchased product.
- DOTCOM POSTAGE generated the highest revenue.
- Customer purchasing activity was concentrated among a small group of highly active customers.
